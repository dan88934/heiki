What is Heiki?

Heiki allows you to create Japanese language vocabulary lists. These lists can be created from an image or a document. For example, if you inputted the example document below, you would receive the example vocabulary list.

Each row contains the original word from the text, the Hiragana reading (if applicable), and the corrosponding English translations for the term.
